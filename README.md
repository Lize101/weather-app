# Weather App

A 7 day London weather forecast application, that displays live data fetched from WeatherAPI.

## Features:
- 7 day weather forecast with dates, icons, temperatures and conditions
- Responsive design across mobile and desktop
- Error handling for API requests

## Tech:
- Vite
- WeatherAPI
- VanillaJS
- HTML
- CSS

## Run the app locally: 
1. Clone the repo from GitHub
2. Run `npm install`
3. Add your API key to .env file as `VITE_API_KEY=yourkey`
4. Run `npm run dev`
